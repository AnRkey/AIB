Contribute to AIB

Thank you for your interest in contributing to AIB! This project welcomes contributions from everyone. Here's how you can help:

Ways to Contribute

  • Report bugs - If you find a bug, please report it using the GitHub issue tracker
  • Suggest features - Have an idea for a new feature? Share it on GitHub
  • Submit code - Pull requests are welcome for bug fixes or new features
  • Improve documentation - Help make the documentation clearer and more comprehensive
  • Share feedback - Let us know how we can improve the application

Getting Started

  1. Fork the repository on GitHub
  2. Clone your fork locally:
    git clone https://github.com/your-username/AIB.git
  3. Install dependencies:
    cd AIB
    npm install
  4. Start the development version:
    npm run dev

Code Contributions

Pull Request Process

  1. Create a branch for your feature or fix
  2. Make your changes, following the project's coding style
  3. Write or update tests if necessary
  4. Update documentation if needed
  5. Submit a pull request to the main repository

Coding Guidelines

  • Follow existing code style and patterns
  • Comment your code where appropriate
  • Keep pull requests focused on a single change
  • Write clean, readable code

Getting Help

If you need help or have questions about contributing, please:

  • Open an issue on GitHub with your question
  • Contact the project maintainers
