AIB Help

Getting Started

AIB is a desktop application that provides a convenient way to access various AI services in a tabbed browser interface. It offers a simple, clean interface optimized for AI interactions.

Using AIB

  1. Click on one of the AI service buttons in the left sidebar (ChatGPT, Claude, Gemini, or Grok) to open that service in a new tab.
  2. Use the tab bar to switch between different AI services.
  3. The refresh button allows you to reload the current tab.
  4. The "AOT" button toggles "Always on Top" mode, keeping the window visible while working in other applications.
  5. The "+Window" button opens a new instance of AIB.

Features

  • Tab-based browsing - Open multiple AI services at once and easily switch between them
  • "Always on Top" mode - Keep the application visible while working in other windows
  • Multiple instances - Launch multiple windows of the application
  • Settings - Configure microphone, always-on-top behavior, proxy settings, and more

Shortcuts

  • F12 - Toggle Developer Tools (if enabled)
  • Ctrl+Click or Middle Click on links - Open in external browser

Troubleshooting

If a service isn't loading correctly:

  1. Try clicking the refresh button in the toolbar
  2. Check your internet connection
  3. Make sure you have an account for the service you're trying to use
  4. If using a proxy, verify your proxy settings

Audio or microphone not working:

  1. Check the audio settings in the Settings menu
  2. Make sure your microphone is properly connected and not muted
  3. Verify you've granted microphone permissions to the application
